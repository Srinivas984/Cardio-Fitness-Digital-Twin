📱 TO SEND DATA FROM APPLE WATCH:

1. Create iPhone Shortcut (see QUICK_SHORTCUT.md)
2. Get your laptop IP:
   ipconfig | findstr IPv4
   (look for 192.168.x.x)

3. Update Shortcut URL:
   POST http://<YOUR_IP>:5000/api/upload

4. Send this JSON:
   {
     "heart_rate": <value>,
     "steps": <value>,
     "active_energy": <value>
   }

5. Shortcut runs every 1 minute automatically
